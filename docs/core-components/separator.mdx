## <Highlight color='#1C5AEC'>설치</Highlight>

커맨드라인에서 컴포넌트를 설치하세요.

```bash
npm install @radix-ui/react-separator
```

## <Highlight color='#1C5AEC'>구성</Highlight>

모든 구성요소를 가져와서 조합하세요.

```jsx
import * as Separator from "@radix-ui/react-separator";

export default () => <Separator.Root />;
```
