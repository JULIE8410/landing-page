
## <Highlight color='#1C5AEC'>구성</Highlight>

컴포넌트를 가져옵니다.

```jsx
import * as Toolbar from "@radix-ui/react-toolbar";

export default () => (
  <Toolbar.Root>
    <Toolbar.Button />
    <Toolbar.Separator />
    <Toolbar.Link />
    <Toolbar.ToggleGroup>
      <Toolbar.ToggleItem />
    </Toolbar.ToggleGroup>
  </Toolbar.Root>
);
```

## <Highlight color='#1C5AEC'>예</Highlight>

#### 다른 primitives와 함께 사용

`Dialog`, `AlertDialog`, `Popover`, `DropdownMenu`와 같이 `Trigger` 부분을 노출하는 모든 프리미티브는 `asChild prop`을 사용하여 toolbar 내에서 구성할 수 있습니다.

`DropdownMenu` primitive를 사용한 예는 다음과 같습니다.

```jsx
import * as Toolbar from "@radix-ui/react-toolbar";
import * as DropdownMenu from "@radix-ui/react-dropdown-menu";

export default () => (
  <Toolbar.Root>
    <Toolbar.Button>Action 1</Toolbar.Button>
    <Toolbar.Separator />
    <DropdownMenu.Root>
      <Toolbar.Button asChild>
        <DropdownMenu.Trigger>Trigger</DropdownMenu.Trigger>
      </Toolbar.Button>
      <DropdownMenu.Content>…</DropdownMenu.Content>
    </DropdownMenu.Root>
  </Toolbar.Root>
);
```
