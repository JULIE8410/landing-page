## <Highlight color='#1C5AEC'>예</Highlight>

#### 수직 방향

Vertical slider prop을 사용하여 수직 슬라이더를 만듭니다.

```jsx
// index.jsx
import * as Slider from "@radix-ui/react-slider";
import "./styles.css";

export default () => (
  <Slider.Root
    className="SliderRoot"
    defaultValue={[50]}
    orientation="vertical"
  >
    <Slider.Track className="SliderTrack">
      <Slider.Range className="SliderRange" />
    </Slider.Track>
    <Slider.Thumb className="SliderThumb" />
  </Slider.Root>
);
```

```css
/* styles.css */
```

#### 범위 만들기

여러 개의 thumb와 값을 추가하여 range slider를 만듭니다.

```jsx
import * as Slider from "@radix-ui/react-slider";

export default () => (
  <Slider.Root defaultValue={[25, 75]}>
    <Slider.Track>
      <Slider.Range />
    </Slider.Track>
    <Slider.Thumb />
    <Slider.Thumb />
  </Slider.Root>
);
```

#### Step 크기 정의

Step prop을 사용하여 스텝핑 간격을 늘립니다.

```jsx
import * as Slider from "@radix-ui/react-slider";

export default () => (
  <Slider.Root defaultValue={[50]} step={10}>
    <Slider.Track>
      <Slider.Range />
    </Slider.Track>
    <Slider.Thumb />
  </Slider.Root>
);
```

#### Thumb 오버랩 방지

`minStepsBetweenThumbs`를 사용하여 같은 값의 thumb를 피합니다.

```jsx
import * as Slider from "@radix-ui/react-slider";

export default () => (
  <Slider.Root defaultValue={[25, 75]} step={10} minStepsBetweenThumbs={1}>
    <Slider.Track>
      <Slider.Range />
    </Slider.Track>
    <Slider.Thumb />
    <Slider.Thumb />
  </Slider.Root>
);
```

## <Highlight color='#1C5AEC'>사용자 정의 API</Highlight>

원시적인 부분을 사용자 자신의 구성요소로 추상화하여 사용자 정의 API를 만듭니다.

#### 모든 부분 추상화

이 예제에서는 자체 닫힘 요소로 사용할 수 있도록 모든 `Slider` 부분을 추상화합니다.

#### 사용

```jsx
import { Slider } from "./your-slider";

export default () => <Slider defaultValue={[25]} />;
```

#### 구현

```jsx
// your-slider.jsx
import * as SliderPrimitive from "@radix-ui/react-slider";

export const Slider = React.forwardRef((props, forwardedRef) => {
  const value = props.value || props.defaultValue;

  return (
    <SliderPrimitive.Slider {...props} ref={forwardedRef}>
      <SliderPrimitive.Track>
        <SliderPrimitive.Range />
      </SliderPrimitive.Track>
      {value.map((_, i) => (
        <SliderThumb key={i} />
      ))}
    </SliderPrimitive.Slider>
  );
});
```

## <Highlight color='#1C5AEC'>주의사항</Highlight>

### <Highlight color='#1C5AEC'>마우스 이벤트가 실행되지 않음</Highlight>

구현 중에 직면한 제한 때문에 다음 예제가 예상대로 작동하지 않고 `onMouseDown` 및 `onMouseUp` 이벤트 핸들러가 실행되지 않습니다:

```jsx
<Slider.Root
  onMouseDown={() => console.log("onMouseDown")}
  onMouseUp={() => console.log("onMouseUp")}
>
  …
</Slider.Root>
```

대신 포인터 이벤트를 사용하는 것을 권장합니다.(예:`onPointerDown`, `onPointerUp`).
위의 제한과 상관없이 이러한 이벤트는 모든 포인터 입력 유형(마우스, 터치, 펜 등)에 대해 실행되므로 크로스 플랫폼/장치 처리에 더 적합합니다.
