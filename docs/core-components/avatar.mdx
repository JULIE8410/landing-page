## <Highlight color='#1C5AEC'>설치</Highlight>

커맨드라인에서 컴포넌트를 설치하세요.

```bash
npm install @radix-ui/react-avatar
```

## <Highlight color='#1C5AEC'>구성</Highlight>

모든 파트를 가져와서 조합하세요.

```jsx
import * as Avatar from "@radix-ui/react-avatar";
export default () => (
  <Avatar.Root>
    <Avatar.Image />
    <Avatar.Fallback />
  </Avatar.Root>
);
```

## <Highlight color='#1C5AEC'>API 참조</Highlight>

### <Highlight color='#1C5AEC'>Root</Highlight>

Avatar의 모든 부분을 포함합니다.

<PropTable api={avatarAPI} category="root" />

### <Highlight color='#1C5AEC'>Image</Highlight>

렌더링할 이미지입니다. 기본적으로 로드된 경우에만 렌더링됩니다.
제어가 더 필요한 경우 `onLoadingStatusChange` 핸들러를 사용할 수 있습니다.

<PropTable api={avatarAPI} category="image" />

### <Highlight color='#1C5AEC'>Fallback</Highlight>

이미지가 로드되지 않았을 때 렌더링하는 요소입니다.
이는 로드 중이거나 오류가 발생한 경우를 의미합니다.
로드 중에 플래시가 나타나면 렌더링을 지연시키는 `delayMs` prop을 제공하여
연결 속도가 느린 사용자만 렌더링할 수 있습니다.
자세한 제어 방법은 `Avatar.Image`의 `onLoadingStatusChange` 핸들러를 사용하십시오

<PropTable api={avatarAPI} category="fallback" />

## <Highlight color='#1C5AEC'>예</Highlight>

#### 도구 설명으로 클릭 가능한 Avatar

추가 정보를 표시하기 위해 Tooltip으로 아바타를 구성할 수 있습니다.

```jsx
import * as Avatar from "@radix-ui/react-avatar";
import * as Tooltip from "@radix-ui/react-tooltip";

export default () => (
  <Tooltip.Root>
    <Tooltip.Trigger>
      <Avatar.Root>…</Avatar.Root>
    </Tooltip.Trigger>

    <Tooltip.Content side="top">
      Tooltip content
      <Tooltip.Arrow />
    </Tooltip.Content>
  </Tooltip.Root>
);
```
