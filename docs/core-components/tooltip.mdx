# Tooltip

요소가 키보드 포커스를 받거나 마우스가 요소 위를 이동할 때 요소와 관련된 정보를 표시하는 팝업입니다.

import React from "react";
import * as Tooltip from "@radix-ui/react-tooltip";
import { PlusIcon } from "@radix-ui/react-icons";
import "./styles.css";

const TooltipDemo = () => {
  return (
    <Tooltip.Provider>
      <Tooltip.Root>
        <Tooltip.Trigger asChild>

          <Tooltip.Content className="TooltipContent" sideOffset={5}>
            Add to library
            <Tooltip.Arrow className="TooltipArrow" />
          </Tooltip.Content>
        </Tooltip.Portal>
      </Tooltip.Root>
    </Tooltip.Provider>
  );
};

export default TooltipDemo;

## <Highlight color='#1C5AEC'>예</Highlight>

#### 전역 구성

`Provider`를 사용하여 `delayDuration` 및 `skipDelayDuration`을 전역적으로 컨트롤합니다.

```jsx
import * as Tooltip from "@radix-ui/react-tooltip";

export default () => (
  <Tooltip.Provider delayDuration={800} skipDelayDuration={500}>
    <Tooltip.Root>
      <Tooltip.Trigger>…</Tooltip.Trigger>
      <Tooltip.Content>…</Tooltip.Content>
    </Tooltip.Root>
    <Tooltip.Root>
      <Tooltip.Trigger>…</Tooltip.Trigger>
      <Tooltip.Content>…</Tooltip.Content>
    </Tooltip.Root>
  </Tooltip.Provider>
);
```

#### 즉시 보여주기

`delayDuration` prop를 사용하여 tooltip이 열리는 데 걸리는 시간을 제어합니다.

```jsx
import * as Tooltip from "@radix-ui/react-tooltip";

export default () => (
  <Tooltip.Root delayDuration={0}>
    <Tooltip.Trigger>…</Tooltip.Trigger>
    <Tooltip.Content>…</Tooltip.Content>
  </Tooltip.Root>
);
```

#### 비활성화된 버튼에서 도구 설명 표시

비활성화된 버튼은 이벤트를 발생시키지 않으므로 다음 작업을 수행해야 합니다:

- `Trigger`를 `span`으로 렌더링합니다..
- `button`에 `pointerEvents`가 없는지 확인합니다.

```jsx
import * as Tooltip from "@radix-ui/react-tooltip";

export default () => (
  <Tooltip.Root>
    <Tooltip.Trigger asChild>
      <span tabIndex={0}>
        <button disabled style={{ pointerEvents: "none" }}>
          …
        </button>
      </span>
    </Tooltip.Trigger>
    <Tooltip.Content>…</Tooltip.Content>
  </Tooltip.Root>
);
```

#### 콘텐츠 사이즈 제한

트리거 너비와 일치하도록 콘텐츠의 너비를 제한할 수도 있고, 뷰포트를 초과하지 않도록 높이를 제한할 수도 있습니다.
이를 지원하기 위해 `--radix-tooltip-trigger-width` 및 `--radix-tooltip-content-available-height`와 같은 여러 CSS 사용자 지정 속성을 노출합니다.
이를 사용하여 콘텐츠 차원을 제한합니다.

```jsx
// index.jsx
import * as Tooltip from "@radix-ui/react-tooltip";
import "./styles.css";

export default () => (
  <Tooltip.Root>
    <Tooltip.Trigger>…</Tooltip.Trigger>
    <Tooltip.Portal>
      <Tooltip.Content className="TooltipContent" sideOffset={5}>
        …
      </Tooltip.Content>
    </Tooltip.Portal>
  </Tooltip.Root>
);
```

```jsx
/* styles.css */
.TooltipContent {
  width: var(--radix-tooltip-trigger-width);
  max-height: var(--radix-tooltip-content-available-height);
}
```

#### 원본 인식 애니메이션

CSS 사용자 지정 속성 `--radix-tooltip-content-transform-origin`을 노출합니다.
이를 사용하여 `side`, `sideOffset`, `align`, `alignOffset` 및 충돌을 기준으로 계산된 원본에서 컨텐츠를 애니메이션화합니다.

```jsx
// index.jsx
import * as Tooltip from "@radix-ui/react-tooltip";
import "./styles.css";

export default () => (
  <Tooltip.Root>
    <Tooltip.Trigger>…</Tooltip.Trigger>
    <Tooltip.Content className="TooltipContent">…</Tooltip.Content>
  </Tooltip.Root>
);
```

```css
/* styles.css */
.TooltipContent {
  transform-origin: var(--radix-tooltip-content-transform-origin);
  animation: scaleIn 0.5s ease-out;
}

@keyframes scaleIn {
  from {
    opacity: 0;
    transform: scale(0);
  }
  to {
    opacity: 1;
    transform: scale(1);
  }
}
```

#### 충돌 인식 애니메이션

`data-side`와 `data-align` 속성을 노출합니다.
그 값은 충돌을 반영하기 위해 런타임에 변경됩니다. 충돌 및 방향 인식 애니메이션을 만들 때 사용합니다.

```jsx
// index.jsx
import * as Tooltip from "@radix-ui/react-tooltip";
import "./styles.css";

export default () => (
  <Tooltip.Root>
    <Tooltip.Trigger>…</Tooltip.Trigger>
    <Tooltip.Content className="TooltipContent">…</Tooltip.Content>
  </Tooltip.Root>
);
```

```css
/* styles.css */
.TooltipContent {
  animation-duration: 0.6s;
  animation-timing-function: cubic-bezier(0.16, 1, 0.3, 1);
}
.TooltipContent[data-side="top"] {
  animation-name: slideUp;
}
.TooltipContent[data-side="bottom"] {
  animation-name: slideDown;
}

@keyframes slideDown {
  from {
    opacity: 0;

  }

    opacity: 1;

  }
}

  from {
    opacity: 0;
    transform: translateY(10px);
  }

  }

```

## <Highlight color='#1C5AEC'>사용자 정의 API</Highlight>

원시적인 부분을 사용자 고유의 컴포넌트로 추상화하여 사용자 정의 API를 만듭니다.

#### Tooltip 추상화 및 content prop 소개

이 예제에서는 모든 `Tooltip` 부분을 추상화하고 새로운 `content` prop을 소개합니다.

#### 사용

```jsx
import { Tooltip } from "./your-tooltip";

export default () => (
  <Tooltip content="Tooltip content">
    <button>Tooltip trigger</button>
  </Tooltip>
);
```

#### 구현

```jsx
// your-tooltip.jsx
import React from "react";
import * as TooltipPrimitive from "@radix-ui/react-tooltip";

export function Tooltip({
  children,
  content,
  open,
  defaultOpen,
  onOpenChange,
  ...props
}) {
  return (
    <TooltipPrimitive.Root
      open={open}
      defaultOpen={defaultOpen}
      onOpenChange={onOpenChange}
    >
      <TooltipPrimitive.Trigger asChild>{children}</TooltipPrimitive.Trigger>
      <TooltipPrimitive.Content side="top" align="center" {...props}>
        {content}
        <TooltipPrimitive.Arrow width={11} height={5} />
      </TooltipPrimitive.Content>
    </TooltipPrimitive.Root>
  );
}
```
