## <Highlight color='#1C5AEC'>설치</Highlight>

커맨드라인에서 컴포넌트를 설치하세요.

```bash
npm install @radix-ui/react-radio-group
```

## <Highlight color='#1C5AEC'>구성</Highlight>

모든 구성요소를 가져와서 조합하세요.

```jsx
import * as RadioGroup from "@radix-ui/react-radio-group";

export default () => (
  <RadioGroup.Root>
    <RadioGroup.Item>
      <RadioGroup.Indicator />
    </RadioGroup.Item>
  </RadioGroup.Root>
);
```

## <Highlight color='#1C5AEC'>API 참조</Highlight>

### <Highlight color='#1C5AEC'>Root</Highlight>

Radio group의 모든 부분을 포함합니다.

<PropTable api={radioGroupAPI} category="root" />
<DataAttributeTable api={radioGroupAPI} category="root" />

### <Highlight color='#1C5AEC'>Item</Highlight>

체크할 수 있는 그룹의 항목입니다. 이벤트가 올바르게 전파되는지 확인하기 위해 `form` 내에서 사용될 때 `input`도 렌더링됩니다.

<PropTable api={radioGroupAPI} category="item" />
<DataAttributeTable api={radioGroupAPI} category="item" />

### <Highlight color='#1C5AEC'>Indicator</Highlight>

라디오 항목이 체크 상태일 때 렌더링합니다. 이 요소를 직접 스타일링하거나, 아이콘을 넣을 래퍼로 사용하거나 둘 다 입력할 수 있습니다.

<PropTable api={radioGroupAPI} category="indicator" />
<DataAttributeTable api={radioGroupAPI} category="indicator" />

## <Highlight color='#1C5AEC'>접근성</Highlight>

[`Radio Group WAI-ARIA design pattern`](https://www.w3.org/WAI/ARIA/apg/patterns/radio/)을 준수하며,
[`roving tabindex`](https://www.w3.org/WAI/ARIA/apg/patterns/radio/examples/radio/)를 사용하여 radio item 간 포커스 이동을 관리합니다.
