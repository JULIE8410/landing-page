## <Highlight color='#1C5AEC'>설치</Highlight>

커맨드라인에서 컴포넌트를 설치하세요.

```bash
npm install @radix-ui/react-hover-card
```

## <Highlight color='#1C5AEC'>구성</Highlight>

모든 구성요소를 가져와서 조합하세요.

```jsx
import * as HoverCard from "@radix-ui/react-hover-card";

export default () => (
  <HoverCard.Root>
    <HoverCard.Trigger />
    <HoverCard.Portal>
      <HoverCard.Content>
        <HoverCard.Arrow />
      </HoverCard.Content>
    </HoverCard.Portal>
  </HoverCard.Root>
);
```

## <Highlight color='#1C5AEC'>예</Highlight>

#### 즉시 표시

`openDelay` prop을 사용하여 호버 카드가 열리는 데 걸리는 시간을 제어합니다.

```jsx
import * as HoverCard from "@radix-ui/react-hover-card";

export default () => (
  <HoverCard.Root openDelay={0}>
    <HoverCard.Trigger>…</HoverCard.Trigger>
    <HoverCard.Content>…</HoverCard.Content>
  </HoverCard.Root>
);
```

#### 콘텐츠 사이즈 제한

트리거 너비와 일치하도록 콘텐츠의 너비를 제한할 수도 있고, 뷰포트를 초과하지 않도록 높이를 제한할 수도 있습니다.
이를 지원하기 위해 `--radix-hover-card-trigger-width` 및 `--radix-hover-card-content-available-height`와 같은 여러 CSS 사용자 지정 속성을 노출합니다. 이를 사용하여 콘텐츠 차원을 제한합니다.

```jsx
// index.jsx
import * as HoverCard from "@radix-ui/react-hover-card";
import "./styles.css";

export default () => (
  <HoverCard.Root>
    <HoverCard.Trigger>…</HoverCard.Trigger>
    <HoverCard.Portal>
      <HoverCard.Content className="HoverCardContent" sideOffset={5}>
        …
      </HoverCard.Content>
    </HoverCard.Portal>
  </HoverCard.Root>
);
```

```css
/* styles.css */
.HoverCardContent {
  width: var(--radix-hover-card-trigger-width);
  max-height: var(--radix-hover-card-content-available-height);
}
```

#### 원본 인식 애니메이션

CSS 사용자 지정 속성 `--radix-hover-card-content-transform-origin`을 노출합니다.
이를 사용하여 `side`, `sideOffset`, `align`, `alignOffset` 및 충돌을 기준으로 계산된 원본에서 컨텐츠를 애니메이션화합니다.

```jsx
// index.jsx
import * as HoverCard from "@radix-ui/react-hover-card";
import "./styles.css";

export default () => (
  <HoverCard.Root>
    <HoverCard.Trigger>…</HoverCard.Trigger>
    <HoverCard.Content className="HoverCardContent">…</HoverCard.Content>
  </HoverCard.Root>
);
```

```css
/* styles.css */
.HoverCardContent {
  transform-origin: var(--radix-hover-card-content-transform-origin);
  animation: scaleIn 0.5s ease-out;
}

@keyframes scaleIn {
  from {
    opacity: 0;
    transform: scale(0);
  }
  to {
    opacity: 1;
    transform: scale(1);
  }
}
```

#### 충돌 인식 애니메이션

`data-side`와 `data-align` 속성을 사용합니다.
이 값은 충돌을 반영하기 위해 런타임에 변경되며, 충돌 및 방향 인식 애니메이션을 만들 때 사용합니다.

```jsx
// index.jsx
import * as HoverCard from "@radix-ui/react-hover-card";
import "./styles.css";

export default () => (
  <HoverCard.Root>
    <HoverCard.Trigger>…</HoverCard.Trigger>
    <HoverCard.Content className="HoverCardContent">…</HoverCard.Content>
  </HoverCard.Root>
);
```

```css
/* styles.css */
.HoverCardContent {
  animation-duration: 0.6s;
  animation-timing-function: cubic-bezier(0.16, 1, 0.3, 1);
}
.HoverCardContent[data-side="top"] {
  animation-name: slideUp;
}
.HoverCardContent[data-side="bottom"] {
  animation-name: slideDown;
}

@keyframes slideUp {
  from {
    opacity: 0;
    transform: translateY(10px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

@keyframes slideDown {
  from {
    opacity: 0;
    transform: translateY(-10px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```
